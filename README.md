# Sales Analytics System

## Overview
This project is a Python-based Sales Analytics System developed as part of Module 3 – Python Programming Assignment.

The system:
- Reads and cleans messy sales transaction data
- Performs sales analytics and customer analysis
- Fetches product data from an external API
- Enriches sales data with API information
- Generates a comprehensive text-based sales report
